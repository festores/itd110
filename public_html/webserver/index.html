Vocabulary

HTML Terms

Internet
a global computer network providing a variety of information and communication facilities, consisting of interconnected networks using standardized communication protocols
TCP/IP
Transmission Control Protocol/Internet Protocol a language a computer uses to access the internet.
LAN
a local area network is a computer network within a small geographical area such as a home, school, computer laboratory, or office buildings
IP address
a unique string of numbers separated by periods that identifies each computer using the Internet Protocol to communicate over a network
HTTP
HyperText Transfer Protocol is an application-layer protocol used primarily on the World Wide Web.
web server
a system that delivers content or services to end users over the internet.
web client
a client-side component within the Java 2 Platform Enterprise Edition (J2EE), a distributed multi-tiered application model used for building and developing enterprise applications.
